Conversor de Arquivos

Descrição

O Conversor de Arquivos é uma aplicação web desenvolvida com Flask, que oferece uma maneira rápida e eficiente de converter arquivos entre diversos formatos. Ele suporta as seguintes conversões:

  • PDF para Word (.docx)
  • Word (.docx) para PDF
  • PDF para Excel (.csv)
  • Excel (.xls/.xlsx) para PDF

A interface é simples e intuitiva, permitindo que você faça upload dos arquivos e realize as conversões em poucos cliques. Ao abrir o arquivo baixado, você encontrará um executável chamado "Conversor", o qual basta abrir para iniciar a aplicação.

Nota importante: Não remova a pasta chamada "Templates", pois ela contém os arquivos essenciais para exibir a interface web e realizar as conversões corretamente.
